## üîÅ 4. –¶–∏–∫–ª—ã –∏ Range ‚Äî –≤–∞–∂–Ω—ã–µ –Ω—é–∞–Ω—Å—ã

Go –∏–º–µ–µ—Ç —Ç–æ–ª—å–∫–æ –æ–¥–∏–Ω —Ü–∏–∫–ª: `for`. –ù–µ—Ç `while` –∏ `foreach`.

```go

    go func() {
        fmt.Println(i, n) // ‚ö†Ô∏è –∑–∞—Ö–≤–∞—Ç—ã–≤–∞–µ—Ç –ö–û–ü–ò–Æ i –∏ n (–æ–¥–Ω–æ –∏ —Ç–æ –∂–µ –¥–ª—è –≤—Å–µ—Ö –≥–æ—Ä—É—Ç–∏–Ω)
    }()

```

üß® –ü—Ä–æ–±–ª–µ–º–∞: `range` —Å–æ–∑–¥–∞—ë—Ç **–Ω–æ–≤—É—é –∫–æ–ø–∏—é –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π** –Ω–∞ –∫–∞–∂–¥–æ–π –∏—Ç–µ—Ä–∞—Ü–∏–∏.  
–ï—Å–ª–∏ –∑–∞—Ö–≤–∞—Ç–∏—Ç—å –µ—ë –≤ –∑–∞–º—ã–∫–∞–Ω–∏–µ, —Ç–æ –≤—Å–µ –≥–æ—Ä—É—Ç–∏–Ω—ã —É–≤–∏–¥—è—Ç –æ–¥–Ω–æ –∏ —Ç–æ –∂–µ –∑–Ω–∞—á–µ–Ω–∏–µ.

‚úÖ –ü—Ä–∞–≤–∏–ª—å–Ω—ã–π —Å–ø–æ—Å–æ–±:

```go

    i, n := i, n // —Å–æ–∑–¥–∞—ë–º –Ω–æ–≤—ã–µ –∫–æ–ø–∏–∏ –≤ –æ–±–ª–∞—Å—Ç–∏ –≤–∏–¥–∏–º–æ—Å—Ç–∏
```

–∏–ª–∏


üìå –¢–∞–∫–∂–µ:  
- `for range` –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç **–∫–æ–ø–∏—é –∑–Ω–∞—á–µ–Ω–∏—è**, –∞ –Ω–µ —Å—Å—ã–ª–∫—É.
